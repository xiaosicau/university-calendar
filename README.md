# 校历助手

一个基于PyQt5开发的校历桌面应用程序，支持自定义导入校历和课表，提供系统托盘、开机自启动、课程表、上课提醒等功能。适用于各类高校教师和学生使用。

**开发者：肖诗顺 - 四川农业大学经济学院金融系**  
**本程序为开源程序，供教师和学生免费使用**

> 提示：用户可通过导入向导自定义本校校历和课表数据

## 运行方式

### 方式一：直接运行可执行文件（推荐）

双击 `校历助手.exe` 即可运行，无需安装Python环境。

### 方式二：源码运行

1. 安装依赖：
   ```
   pip install PyQt5 openpyxl
   ```

2. 运行程序：
   ```
   python sicau_calendar.py
   ```

## 功能特性

- 显示学校校历和重要日期（开学、放假、考试、节日等）
- 课程表管理，支持手动添加和Excel导入
- 上课前30分钟闹钟提醒
- 上课前一天弹窗提醒
- 系统托盘后台运行
- 支持开机自启动
- 支持自定义导入校历和课表数据

## 数据导入

首次运行时会启动导入向导，您可以：

1. 设置学校名称和学年
2. 配置学期开始/结束日期
3. 添加重要日期（开学、放假、考试等）
4. 添加课程表（支持手动添加或从Excel导入）

### Excel课表格式

从Excel导入课程时，表格列顺序应为：

| 课程名 | 教师 | 教室 | 星期 | 节次 | 周次 |
|--------|------|------|------|------|------|
| 高等数学 | 张老师 | 101 | 周一 | 1-2 | 1-16 |

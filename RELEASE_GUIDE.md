# GitHub发布指南

## 发布预编译版本

本项目提供预编译版本供用户直接下载使用，无需安装Python环境。

### 发布流程

1. **准备发布资产**
   - 编译好的可执行文件（校历助手.exe）
   - 项目源代码压缩包
   - 版本说明文档

2. **创建Release**
   - 在GitHub仓库中点击"Releases"
   - 点击"Draft a new release"
   - 输入标签（如v2025-2026）
   - 输入发布标题（如"2025-2026学年版本"）
   - 在描述中说明此版本的特性和改进

3. **上传资产**
   - 将编译好的 `校历助手.exe` 文件拖拽到"Attach binaries by dropping them here or selecting them."区域
   - 也可上传其他相关文件，如使用说明等

### 发布内容

每次发布应包含：

- **预编译版本**：`校历助手.exe` - 为Windows用户提供的可执行文件
- **源代码**：完整的项目源代码，便于开发者参考和贡献
- **版本说明**：详细的版本更新日志

### 文件说明

```
校历助手_v2025-2026/
├── 校历助手.exe              # 预编译的可执行文件
├── README.md               # 项目说明
├── USAGE.md                # 使用说明
├── VERSION_INFO.md         # 版本信息
└── source/                 # 源代码文件夹
    ├── sicau_calendar.py
    ├── requirements.txt
    └── data/
        └── calendar_data.json
```

### 更新策略

- 当四川农业大学或其他高校公布新的校历后，会及时发布适配新校历的新版本
- 预计每年发布春季和秋季两个学期的新版本
- 用户可在GitHub Releases页面订阅更新通知

### 用户指南

**对于普通用户：**
- 下载预编译的 `.exe` 文件即可直接使用
- 无需安装Python或其他依赖

**对于开发者：**
- 可下载源代码进行定制或贡献
- 参考 `requirements.txt` 安装依赖

### 注意事项

- 预编译版本为Windows平台
- 源代码版本支持跨平台使用
- 如有问题请在Issues中报告
